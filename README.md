# DiVA
DiVA stands for DFactory Image Retrieval. This repository showcases a Proof of Concept (PoC) for a sophisticated image search engine tailored for private datasets. It is designed to efficiently retrieve visually similar images using state-of-the-art deep learning models.

# Project Setup Instructions
This project was developed using WSL (Ubuntu 24.04.2 LTS on Windows 11) and Python v3.12.3. Before you begin, ensure you have UV installed as your Python package manager.

## Prerequisites

- **Python v3.12.3**
- **UV**

## Setup steps

1. **Initialize the project and install dependencies**
    ```
    sh scripts/setup.sh
    ```

2. **Run NAS path extractor**
    ```
    sh scripts/run_executor.sh --path services/nas/executor.py
    ```

3. **Mount the NAS**
    ```
    sh scripts/run_mounter.sh
    ```

4. **Extract client preview data**
    ```
    sh scripts/run_executor.sh --path services/monitoring/extractor.py
    ```

5. **Create FAISS database**
    ```
    sh scripts/run_executor.sh --path utils/zero_shot/classifier.py
    ```

6. **Start the API server**
    ```
    sh scripts/run_server.sh --env development --port 10000
    ```
